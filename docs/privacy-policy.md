# MoneyMind 개인정보처리방침

최종 업데이트: 2026-03-10

## 1. 개요

MoneyMind는 개인용 가계부 앱입니다. 거래 알림, 사용자가 직접 선택한 명세서 파일, 수동 입력 데이터를 기기 내부에 저장하여 가계부 기록과 예산 분석 기능을 제공합니다.

## 2. 수집 및 처리하는 데이터

- 거래 알림 본문
  - 사용자가 안드로이드 설정에서 알림 접근 권한을 허용한 경우에만 처리합니다.
- 사용자가 직접 선택한 파일
  - CSV, XLS/XLSX, PDF 명세서 파일
- 사용자가 직접 입력한 정보
  - 거래 내용, 카테고리, 예산, 계좌, 룰, 메모, 할부 계획

## 3. 이용 목적

- 거래 내역 저장
- 거래 분류 및 카테고리 추천
- 예산 진행도와 월말 정산 계산
- CSV 내보내기

## 4. 외부 전송 및 제3자 제공

기본 앱 구성에는 광고 SDK, 분석 SDK, 외부 동기화 서버가 포함되어 있지 않습니다. 처리한 거래 데이터는 기본적으로 기기 내부에만 저장되며, 사용자가 직접 실행한 CSV 내보내기 외에 자동 외부 전송을 하지 않습니다.

## 5. 보관 기간

앱 내부 데이터는 사용자가 삭제하거나 앱을 제거할 때까지 기기에 보관됩니다. 사용자는 앱의 초기화 기능을 통해 기간별 삭제, 전체 삭제, 공장초기화를 직접 수행할 수 있습니다.

## 6. 권한

- 알림 접근 권한
  - 금융 앱 알림을 가계부 거래로 가져오기 위해 사용합니다.
- 파일 선택 권한
  - 사용자가 직접 선택한 명세서 파일을 읽기 위해 사용합니다.

MoneyMind는 공개 배포용 구성에서 SMS/통화기록 권한을 요청하지 않습니다.

## 7. 백업 정책

MoneyMind는 공개 배포용 기본 설정에서 안드로이드 자동 백업을 사용하지 않습니다. 데이터 보존이 필요하면 사용자가 CSV 내보내기를 통해 직접 백업해야 합니다.

## 8. 사용자 권리 및 문의

사용자는 앱 내 기능을 통해 자신의 데이터를 확인, 내보내기, 삭제할 수 있습니다. 저장소 운영자에게 문의가 필요하면 GitHub 저장소 이슈를 통해 연락할 수 있습니다.

- 저장소: https://github.com/Sialll/NEW-PROJECT

## 9. 변경 사항

본 방침은 앱 기능 또는 배포 정책이 변경되면 함께 업데이트됩니다.
